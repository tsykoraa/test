# Git Nefunguje

```
V gitku sa mi chyby množia,
klonujem, pullujem, no stále sa trápia.
"fatal: refusing to merge unrelated histories,"
kričí mi terminál, s ním aj moje nervy.

Zúfalo hľadám riešenie online,
fóra prezerám, stackoverflow skúsim snáď.
"Rebase, reset, merge --no-ff,"
všetko skúšam, no stále nič, len zmätok a hnev.
```
